Notepad RT

Overview

Notepad RT - the missing code editor for developers and geeks for Windows 8 with rich support for text manipulation and syntax highlighting. Built specifically for Windows 8, Notepad RT matches the features and performance of desktop-class editors.

Notepad RT is now available for purchase in the Windows 8 Store. For a limited time to celebrate our release and Windows 8 general availability the app is 50% off (only $2.49).

Go to the Windows Store

Features

  • Syntax highlighting for HTML, CSS, JavaScript, PHP, Python, Ruby, C/C++, C#, Markdown, and many more languages.
  • Several popular Dark and Light Syntax highlighting themes to choose from.
  • Live syntax checking for JavaScript, CoffeeScript, and CSS.
  • Automatic indention, Line wrapping, and Code folding.
  • Highlighting of matching parentheses and selected line
  • Toggle invisible characters.
  • Drag and drop text using your finger or mouse.
  • Remains responsive even with very large files.
  • Open files directly from the Desktop.
  • Use the Share Charm to send code via IM or Mail.
  • Work with multiple cursors and selections.
  • Fast and Fluid interface at home on a desktop or tablet.
  • Edit any compatible file from the file picker.
